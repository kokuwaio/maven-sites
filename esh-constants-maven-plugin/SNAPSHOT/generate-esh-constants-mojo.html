esh-constants:generate-esh-constants

Full name:

io.kokuwa.edge:esh-constants-maven-plugin:0.1.15-SNAPSHOT:generate-esh-constants

Description:

Mojo for generating constant class from "ESH-INF" folder. The plugin scans for XML files in the input folder and looks for a couple of XPath matching nodes to extract constants strings and writes them to a Java constants class using a Freemarker template.

Attributes:

  • Requires a Maven project to be executed.
  • The goal is thread-safe and supports parallel builds.
  • Since version: 0.1.0.
  • Binds by default to the lifecycle phase: generate-sources.
